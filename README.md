## Run in first time
0. Start [dd_trainee](https://github.com/rumbletomato/dd_trainee) project
1. Build project with gradle
2. `docker-compose build`
3. `docker-compose up -d`

## Run in other times
0. Start [dd_trainee](https://github.com/rumbletomato/dd_trainee) project
1. `docker-compose up -d`

## Stop all
1. `docker-compose down --remove-orphans`
